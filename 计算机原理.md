原码补码反码是计算机计算负数用的一种编码。
计算机中的数在内存中都是以补码表示的, 所以 ^1 等于 -2    0000 0001 => 1111 1110 => 需要将补码转化为原码 => 1111 1101 (反码) => 1000 0010 (原码)
